## Development Setup

To set up your development environment, follow these steps:

```bash
# Clone the repository
git clone https://github.com/dragonsbootcampllc/ReactFlaim.git

# Navigate into the repository
cd ReactFlaim

# Install dependencies using npm or yarn
npm install
# or
yarn
```

## Submitting Changes

Please follow these steps to make contributions:

### Create a Branch

- **Create a new branch** from the main branch using the command:

  ```bash
  git checkout -b feature/your-feature-name
  ```

## Make Your Changes

- **Modify the code** ensuring that it meets our coding standards.
- **Add or update tests** as necessary to ensure new changes are covered.

## Write Meaningful Commit Messages

- Write clear and descriptive **commit messages**, explaining why and what changes you've made. This documentation of your changes helps others understand the purpose of your edits and facilitates the review process.

## Run Tests

Ensure that the complete test suite passes to confirm your changes do not disrupt existing functionalities. Run tests using:

```bash
npm test
```

## Submit a Pull Request

- **Push your changes** to your fork.
- **Submit a pull request** to the main repository.
- Ensure to **fill out the PR template** with a detailed description of your changes.

## Pull Request Checklist

Ensure your pull request adheres to the following guidelines:

- Does the code **adhere to the coding standards**?
- Are new features or changes **covered by tests**?
- Is the **new feature or update documented**, if necessary?
- Is your **commit message clear and informative**?
